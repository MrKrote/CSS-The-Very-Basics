# CSS-The-Very-Basics

> CSS: The Very Basics (Cascading Style Sheets)
>

**What is CSS?**

- CSS is a language for describing how documents are presented visually.

**CSS Rules**

Almost everything you do in CSS follows this basic pattern:

```
selector {
    property: value;
}
```

**Pl.:**

Make all ```<h1>``` elements purple
  
```
h1 {
    color: purple;
}
```
